# env setup
- Create virtual env for dependencies while dev: `python3 -m venv .venv`
- Activate virtual env: `. .venv/bin/activate`
- install deps `pip install -r ./requirements.txt`
- save new deps `pip freeze > requirements.txt`

# run tests
- `python -m pytest`
